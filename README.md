# RC-CAR
This project is a Bluetooth-controlled RC car using an Arduino and a mobile app. It allows wireless control of movement like forward, backward, left, and right.
This project demonstrates a Bluetooth-controlled RC car using an Arduino, motor driver, and Bluetooth module. The Arduino acts as the brain of the system, receiving commands from a smartphone via the Bluetooth module (HC-05).

A mobile app sends control signals such as forward, backward, left, right, and stop. These commands are received by the Arduino through serial communication and processed accordingly. Based on the command, the Arduino controls the motor driver, which drives the motors to move the car in the desired direction.

This project introduces important concepts like wireless communication, motor control, and robotics. It is interactive, easy to build, and a great beginner project for understanding real-time control systems.
